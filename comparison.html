Norman vs GOMS vs KLM

Comparison of HCI Models (Dix et al., Ch. 9 & 12)
Aspect Norman GOMS KLM
Focus High-level user interaction: goal formation, execution, evaluation Expert, routine task performance; structured task decomposition Execution time prediction for low-level actions
Cognitive Scope Perceptual, cognitive, motor stages Structured task steps & choice rules Physical operators only (K,P,H,D,M,R)
Assumes Expert Users? No, includes errors & interpretation Yes, error-free Yes, error-free
Predictive Output Qualitative usability insights Task structure and predicted user behavior Execution time for routine tasks
Components Stages of action: goal → execution → evaluation; gulfs of execution/evaluation Goals, Operators, Methods, Selection rules Operators only: Keystroke, Pointing, Homing, Drawing, Mental, System response
Best Use Diagnosing UI problems; understanding user thought processes Analyzing expert task performance & method comparison Comparing interface efficiency quantitatively
Limitation Not predictive of exact timings Not suitable for novices or errors Does not model planning, goals, or method selection
